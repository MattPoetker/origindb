⚡ InstantDB
Real-time database for modern applications

🍎 macOS & Linux

Install InstantDB with a single command:

curl -sSf https://install.instantdb.com | sh

🪟 Windows (PowerShell)

Install using PowerShell (recommended):

iwr -useb https://install.instantdb.com/windows | iex

🪟 Windows (Command Prompt)

Alternative installation for Command Prompt:

curl -sSf https://install.instantdb.com/windows.bat | cmd
